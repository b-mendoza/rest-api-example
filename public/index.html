Ejemplo de REST API

Endpoints

GET /events

Obtiene todos los eventos disponibles

Ejemplo: Resultado (JSON)

GET /events/:id

Obtiene todos los eventos a partir de un ID

Example: Resultado (JSON)

POST /events

Crea un evento

Cabeceras (Headers)

Content-Type: application/json

Cuerpo de la petición

{
    "topics": "",
    "thumbnail": "/img/tr-3.jpeg",
    "url": "index.html",
    "overrideURL": "",
    "linkType": "",
    "title": "Lorem ipsum,
    "summary": "Lorem ipsum dolor sit amet"
}

PUT /events/:id

Actualiza un evento existe a partir de un ID

Cabeceras (Headers)

Content-Type: application/json

Cuerpo de la petición

{
    "id": "1",
    "topics": "",
    "thumbnail": "/img/tr-3.jpeg",
    "url": "index.html",
    "overrideURL": "",
    "linkType": "",
    "title": "Lorem ipsum",
    "summary": "Lorem ipsum dolor sit amet"
}

DELETE /events/:id

Elimina un evento a partir de un ID
